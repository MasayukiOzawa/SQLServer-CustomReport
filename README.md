# SSMSCustomReport
別名「えろす師匠鰻奢ってくださいレポート」

SQL Server Management Studio のカスタムレポートとして、SQL Server の各種情報をダッシュボードとして表示します。

以下の情報を表示することができます。
- 代表的な sp_configure の設定
- トレースフラグ
- 代表的な待ち事象
- データベースファイル使用状況
- ファイル I/O
- インデックス使用状況
- 欠落しているインデックス
- キャッシュ使用状況
- データキャッシュの比率
- プランキャッシュの比率

取得できる情報は追加中です。

# 動作確認状況
SQL Server 2012 + SQL Server 2014 Management Studio

# 作成環境
